Mortgage Mission Control is a compact dashboard for keeping an eye on an Argentine UVA mortgage.

## Overview

The project turns a recurring personal finance question into a small, glanceable web app. It was built to be fast, simple, and easy to check from any device.

## Technical Stack

- **Frontend**: Static web app
- **Hosting**: Cloudflare Workers Static Assets
- **Focus**: Lightweight personal dashboard

## Result

A practical personal tool hosted at the edge with very little operational overhead.
